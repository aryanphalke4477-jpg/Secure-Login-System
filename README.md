Secure Login System
A Flask-based secure login system implementing password hashing, session
management, and input validation.
1) Features
- User registration
- Secure login/logout
- Password hashing
- Session management
- SQLite database
- Input validation
- Dark UI
2) Technologies Used
- Python
- Flask
- SQLite
- Werkzeug Security
- HTML/CSS
3) Run Project
```bash
py -m pip install -r requirements.txt
py app.py
